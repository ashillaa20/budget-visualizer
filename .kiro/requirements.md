# Expense & Budget Visualizer

Simple expense tracking web application built using HTML, CSS, and Vanilla JavaScript.

## Features
- Add transactions
- Delete transactions
- Total balance calculation
- Spending chart visualization
- Local storage support
- Responsive mobile layout

## Categories
- Food
- Transport
- Fun

## Optional Features
- Custom categories
- Transaction sorting
- Responsive UI improvements

## Technical Stack
- HTML
- CSS
- Vanilla JavaScript
- Chart.js

## Storage
- Browser Local Storage API

## Notes
This project is designed to be lightweight, responsive, and easy to use.
